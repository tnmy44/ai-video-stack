▶ Lowest-friction path to start

  1. Claude / ChatGPT ($0–20) → write the lesson script.
  2. Argil (~$39) → generate the talking-head video from your cloned self.
  3. Adobe Podcast Enhance ($0) → only if you record real footage/demos.
  4. Submagic ($12–23) → captions, b-roll, and visual emphasis.

Start free on #1 and #3, trial #2 and #4, then commit. A full educational pipeline runs ~$50–80/month.
